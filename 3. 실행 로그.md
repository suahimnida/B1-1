
실행 로그 (대화 로그 1개)
: 선택한 업무 과업 시나리오로 10턴 이상 실제 대화를 수행한 전문과, 문제 발생 지점(환각/문맥 끊김 등) 및 수정 결과 요약을 담은 로그
  1) 10턴 이상 대화 로그 + 문맥 유지 검증
      -> 중간에 조건 변경(톤/대상/정책 변경) or 나중에 추가정보 제공(첨부 내용/제약 조건)
           -> 확정된 요구사항을 10턴 내내 일관되게 유지 + 조건 변경이 들어올 시, 변경된 부분만 반영할 시, Pass 판정
                -> 대화 로그를 그대로 덤프하지 말고 가독성을 위해 섹션/번호/요약을 추가 정리 + 원본 로그는 확인할 수 있도록 파일 or 링크 첨부
